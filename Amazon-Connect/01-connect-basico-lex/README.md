# 01 — Amazon Connect Básico + Lex

## 📋 Visão Geral

Implementação de um contact center básico com Amazon Connect integrado ao Amazon Lex para reconhecimento de intenções via texto. O fluxo direciona o cliente para filas de atendimento com base na intenção identificada pelo bot.
