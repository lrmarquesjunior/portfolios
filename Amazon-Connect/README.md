> 💼 Estes projetos refletem experiência real em implementações de contact center na AWS.

## 🧱 Serviços Cobertos

- **Contact Center:** Amazon Connect, Contact Flows, Queues, Routing Profiles
- **IA/NLP:** Amazon Lex V1 e V2, Amazon Polly
- **Compute:** AWS Lambda (Node.js / Python)
- **Banco de Dados:** Amazon DynamoDB
- **Integração:** API Gateway, chamadas a APIs externas
- **Analytics:** Amazon Kinesis Data Streams, S3, Athena

---

## 🔑 Conceitos Demonstrados

- Criação e exportação de Contact Flows
- Integração Lambda dentro de fluxos de atendimento
- Consulta de dados em tempo real via DynamoDB
- Integração com serviços externos (REST APIs)
- Bots conversacionais com Lex V2
- Análise de dados de atendimento com Kinesis
